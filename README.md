## ✏️ 구현 방법 및 어려웠던 점

### Toggle

- 구현 방법
  - 토글 버튼 클릭 여부를 boolean 값으로 관리하기 위해서 useState로 상태 관리하는 방법으로 구현했습니다.
- 어려웠던 점
  - className을 두 개 이상 할당하는 과정에서 문법적인 오류로 어려움이 있었습니다. 구글 서칭을 통해 삼항연산자를 사용하는 방법을 해결했습니다.
- 실행 방법
  - 스위치 버튼 영역 클릭 시 스위치 토글
